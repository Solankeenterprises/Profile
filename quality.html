Manufacturing Case Study
API 6D Class — 24" Gate Valve Body Machining (Cast Iron, 50-Piece Production Run)
Challenge
Machine a 36" gate valve body (Grade CI) requiring tight seat concentricity (±0.02mm) and bore alignment across a complex multi-cavity casting for a confidential API 6D-certified valve OEM. Component required zero leakage at hydrotest pressure.
Our Approach
Deployed CNC VTL (1600mm swing) for body turning, followed by Scharmann HBM for precision bore centreline alignment. Developed custom soft-jaw fixtures for repeatable clamping. Maintained SPC at all critical features throughout the run.
Results Delivered
<strong>±0.015mm concentricity achieved</strong> — exceeding specification. <strong>Zero leakage</strong> on hydrotest across all 50 pieces. Delivered in <strong>4 weeks</strong>. Client has placed 4 consecutive repeat orders for the same component family.
